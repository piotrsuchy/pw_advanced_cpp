# Dokumentacja projektu Sieciowy PacMan

Ten projekt jest implementacją gry PacMan w trybie sieciowym. Dokumentacja pojawiająca się w docs/ jest potrzebna do zaliczenia kursu na uczelni.

Dokumentacja składa się z:

- główne README.md w root projektu
- folder docs/
  - zawierający folder `images/`, czyli zdjęcia użyte do wygenerowania raportu
  - zawierający folder `raports/`, czyli wygenerowane raporty, w `old_version/` - z pierwszego terminu, w `new_version/` - z drugiego terminu
  - folder z raportami zawiera również pliki z komentarzami od prowadzącego, np. `projekt_etap_1_Piotr_Suchy_feedback.txt`
- REPORT.md na podstawie którego generujemy przyrostowo dokumentację - tzn., projekt_etap_2_Piotr_Suchy.pdf ma zawierać to co w projekt_etap_1_Piotr_Suchy.pdf + nowości z etapu 2.
